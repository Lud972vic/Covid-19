## Environnement de développement
* PHP 7.4.16
* Composer
* Symfony CLI

## Vérifier les pré-requis
symfony check:requirements

## Savoir la version de Symfony du projet
php bin/console about

## Compiler les SCSS
npm run build   

## Server interne en arrière plan
symfony serve -d"# Covid-19"
